Html,css y javascript

Son los tres pilares fundamentales del desarrollo web que trabajan juntos para crear páginas web dinámicas e interactivas HTML se encarga de la estructura y el contenido, CSS del diseño y la presentación, y JavaScript de la interactividad y la funcionalidad.

Definiciones

HTML (HyperText Markup Language):
Es el lenguaje de marcado que define la estructura y el contenido de una pagina web. Utilizando etiquetas para delimitar diferentes elemetos, como parrafos, encabezados, imagenes, enlaces, etc
CSS (Cascading Style Sheets):
Es el lenguaje de estilos que se utiliza para controlar la presentacionde una pagina web. Permite definir aspectos como los colores, fuentes, tamaños, márgenes, etc, de los elementos HTML
JavaScript (Program. Lenguaje):
Es un lenguaje de programación que permite agregar funcionalidad interactiva a una pagina web. Permite menipular elementos HTML, responder a eventos del usuario, realizar cálculos, validar formularios, etc
